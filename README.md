## 📦 Templates Included

| Template | Description |
|---|---|
| **Sales Dashboard** | Revenue trends, top customers, KPI cards |
| **P&L Report** | Income statement with waterfall chart |
| **Cash Flow** | Operating, investing, financing flows |
| **Inventory** | Stock levels, low-stock alerts, reorder status |

## 🛠️ Local Development

```bash
git clone https://github.com/INSPIRON-TECH-BD/MANAGER-IO-DASHBOARD-TEMPLATES
cd MANAGER-IO-DASHBOARD-TEMPLATES
npm install
npm run dev
# Open http://localhost:3000
```

## 🏗️ Tech Stack

- **Framework**: Next.js 14 (App Router, Static Export)
- **Styling**: Tailwind CSS + INSPIRON TECH brand tokens
- **Charts**: Recharts
- **Icons**: Phosphor Icons
- **Deploy**: GitHub Actions → GitHub Pages
